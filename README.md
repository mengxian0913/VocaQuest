## 專案概述

VocaQuest 是一個基於命令列介面的單字學習系統，採用完整的物件導向設計原則開發。系統提供單字管理、智能測驗、日誌追蹤等功能，幫助使用者有效記憶與複習英文單字。

### 系統特色

- ✨ **模組化設計**: 採用物件導向設計，各功能模組獨立
- 🎯 **智能測驗**: 隨機四選一選擇題測驗機制
- 📝 **完整日誌**: 記錄所有操作歷史，提供系統狀態追蹤
- 💾 **資料持久化**: 支援檔案自動讀寫，確保資料安全
- 🔧 **易於擴展**: 良好的架構設計，便於添加新功能

## 技術架構

### 核心類別

| 類別           | 職責                   | 負責人 |
| -------------- | ---------------------- | ------ |
| `AlertMsg`     | 訊息封裝與時間戳記管理 | 陳宗佑 |
| `Log`          | 系統日誌記錄與查詢     | 陳宗佑 |
| `Vocabulary`   | 單字資料管理與檔案 I/O | 楊孟憲 |
| `Quiz`         | 測驗邏輯與選項生成     | 李則瑞 |
| `VocaQuestApp` | 主控制器與使用者介面   | 楊孟憲 |

### 物件導向特性應用

- **封裝 (Encapsulation)**: 所有類別採用私有資料成員，提供公有介面
- **組合 (Composition)**: `VocaQuestApp` 包含 `Vocabulary`，`Log` 包含 `AlertMsg`
- **依賴注入**: `Quiz` 透過建構子接收 `Vocabulary` 資料
- **設計模式**: 應用單例模式 (Log)、工廠模式概念 (AlertMsg)

## 系統功能

### 主選單選項

1. **新增單字卡片** - 支援英中對照單字新增
2. **打印所有卡片** - 顯示資料庫中所有單字
3. **刪除單字卡片** - 移除指定的英文單字
4. **查詢單字卡片** - 搜尋特定單字的翻譯
5. **開始考試** - 進入四選一選擇題測驗模式
6. **結束程式** - 儲存資料並安全退出
7. **查詢系統狀態** - 檢視完整操作日誌

### 測驗系統特色

- 🎲 **隨機出題**: 確保每次測驗都有不同的題目組合
- ⚙️ **自訂題數**: 使用者可指定測驗題目數量
- ⚡ **即時回饋**: 答題後立即顯示正確答案與解釋
- 📊 **進度追蹤**: 顯示目前答對題數與測驗進度

## 安裝與執行

### 系統需求

- C++14 以上編譯器 (推薦使用 clang++ 或 g++)
- Make 建置工具
- 支援 ANSI 終端機控制碼的終端機

### 編譯步驟

```bash
# 1. 複製專案
git clone <repository-url>
cd VocaQuest

# 2. 編譯程式
cd src
make

# 3. 執行程式
./vocaquest
```

### 手動編譯

```bash
cd src
clang++ -std=c++14 -Wall -Wextra -o vocaquest \
    main.cpp AlertMsg.cpp Log.cpp Quiz.cpp \
    Vocabulary.cpp VocaQuestApp.cpp
```

## 資料格式

系統使用 `vocData.txt` 儲存單字資料，格式如下：

```
apple 蘋果
banana 香蕉
computer 電腦
programming 程式設計
vocabulary 詞彙
```

每行包含一個英文單字和對應的中文翻譯，以空格分隔。

## 使用範例

### 新增單字

```
請輸入英文: programming
請輸入中文: 程式設計

```

### 開始測驗

```
請輸入題數(最少 1 題 | 最多 5 題): 3

請選擇 apple 的中文:

(1): 香蕉

(3): 電腦
(4): 詞彙

你的選擇: 2
答對了!!
```

## 程式設計亮點

### 演算法優化

- **資料結構選擇**: 使用 `std::map` 提供 O(log n) 查找效率
- **隨機化演算法**: 使用 `std::set` 確保測驗題目不重複
- **記憶體管理**: 適當使用 STL 容器避免手動記憶體管理

### 效能分析

| 操作         | 資料結構               | 時間複雜度 |
| ------------ | ---------------------- | ---------- |
| 單字查詢     | std::map               | O(log n)   |
| 單字新增     | std::map               | O(log n)   |
| 測驗題目生成 | std::set + std::vector | O(k log n) |
| 日誌記錄     | std::vector            | O(1)       |
